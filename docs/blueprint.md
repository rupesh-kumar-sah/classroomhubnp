# **App Name**: ClassroomHub

## Core Features:

- Free Class 12 Notes: Provide comprehensive, syllabus-aligned notes for core Class 12 subjects (Physics, Chemistry, Math, Biology, English).
- Coding Tutorials: Offer interactive coding tutorials for beginners (Python, C++, HTML/CSS/JavaScript).
- Coding Challenges: Present coding challenges with a built-in code editor to test the students abilities.
- Premium Study Guides: Provide detailed, professionally curated notes for entrance exams (JEE, NEET, SAT).
- Progress Tracking: A personalized dashboard to track progress, quiz scores, and bookmarked notes.
- Intelligent Search: Implement a tool-augmented predictive search for notes, videos, and topics; allow results to be filtered by subject and content type.

## Style Guidelines:

- Primary color: Strong blue (#29ABE2) for trustworthiness and stability.
- Background color: Light gray (#F0F2F5) for a clean, distraction-free reading environment.
- Accent color: Bright orange (#FF9933) for call-to-action buttons and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif providing both modern looks and a touch of warmth. Note: currently only Google Fonts are supported.
- Use simple, clear icons for navigation and content categories.
- Clean and intuitive layout with clear separation of free and premium content.
- Subtle animations and transitions to enhance user experience (e.g., page transitions, progress bar updates).
